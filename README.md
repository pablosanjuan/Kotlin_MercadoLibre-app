# App para Mercado Libre

## About this project:
* Android project 100% kotlin
* Clean architecture
* MVVM pattern
* Repository pattern
* Motion Layout
* Lottie
* DataBinding
* ViewBinding
* Material Design
* Coroutines
* Navigation graph
* Dagger
* Dark Mode
* Error Handler
* Retrofit
* Unit Test
